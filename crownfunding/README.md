# Crowdfunding
A crowdfunding website.

# How to use
Please add `config.php` into `crowdfunding/config/` folder.

```php
<?php
# In "config.php"
# DB configuration
define('DB_NAME', 'crowdfunding');
define('DB_USER', 'USERNAME');
define('DB_PASSWORD', 'PASSWORD');
define('DB_HOST', 'localhost');
```
